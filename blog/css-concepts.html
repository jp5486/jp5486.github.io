September 2015

What is the difference between Margin, Border, and Padding?

Margin clears an area around the border.

Border is a border(for lack of a better term. Its pretty self explanatory) that goes around the content and padding.

Padding clears an area around the element.

Look at the imagine below for the box model.
Content central to everything and surrounded by the blank padding space. The border creates a designed box around both the content and padding. Finally, the margin is the space outside the content, padding and margin.

What are the best practices associated with using classes vs. ids?

Best practices call for heavy use of Classes and barely any use of IDs. The reasoning behind this is there will not be many situations where you need to style a specific element and it is a waste of time and resources to code an element only in one place. You can take it to an extreme by looking at atomic design patterns where a single class name is set for every single style modification set. Your code is reusable and pretty much unbreakable with no syntax errors.

What are the differences between relative, absolute, fixed, and static positioning?

Absolute is only affected relative to the next parent element with relative or absolute positioning.
Fixed positioning has the element positioned relative to the browser window.
Static is the default natural positioning. It is used to forcefully remove some unintended positioning effects.
Relative is confusing and means relative to itself. It is similar to static positioning as it starts off not affected by other elements and is then moved accordingly from its static position.

What does it mean to display inline vs using inline block?

When you display inline you will use the left and right margin & padding and allow other elects to be placed on the same horizontal plane.
Display inline block is similar to inline where you can still place elements on the same horizontal plane however it will use the top + bottom margins & padding along with height and width properties.
