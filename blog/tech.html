Regular Expressions

Nov 1, 2015

Regular expressions are a very powerful tool that can be used across languages to help search out patterns in your code.
Below is a summary of how to reference different search patterns:

The . character is a general joker/wildcard

Surrounding letters with brackets will look for those sepecfic letters
[fgh] would look for any of the letters: 'f', 'g', 'h'.

While using a hat ^ inside the brackets will exclude letters from a search
[^ijk] would skip words with the letters 'i' 'j' or 'k' from your search.

Ranges can also be set within brackets:
[4-9] will search for the digits 4 through 9
[^a-g] will search for all letters except a through g

Note that letters are case sensitive
[A-C] will only get the words containing Upper Case letters 'A','B','C'

Searching for multiple occurances of a letter is fairly simple
\x\x\x or x{3} will search for 3 instances of the lower case letter 'x' connected
But there are other ways as well:

A * character represents 0 or more of a charater while
the + character represents 1 or more and makes more sense to use.
B+ looks for 1 or more Upper Case letter B
