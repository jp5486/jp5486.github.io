Enumerable Methods

Oct 9, 2015

Today I will be covering the Enumerable method ".cycle"

An enumerable method can only be called when iterating over all the values stored in a variable by using ".each" or its equivalent.
The ".cycle" method when called, will use the number you pass through it as the number of times to iterate over all the values.
Now lets see an example.

Here we are iterating over all values in the my_hash 3 times and adding 1 to each of those values. Since the cycles had completed properly and without interruption, nil was passed at the end.
Note: if you do not pass a number parameter when calling .cycle, it will not stop unless you send a hard break singal through your terminal.
